HTML Base structure
Element Name Description
<html></html> HTML element HTML root that includes everything that is inside the HTML
<head></head> Head element It contains the metadata of an HTML, and it's not visible
<body></body> Body element Contains all the visible part of a page
HTML Metadata
Element Name Description
<link> Link element To link external resources such as CSS, fonts, favicons, etc
<meta> Meta element Contains meta that cannot be represented by base, link, script, style or title
<title> Title element Title of the page, it shows in the browser title bar and tab
HTML structure
Element Name Description
<address></address> Address element Includes contact information person, people, and organizations
<article></article> Article element Represents a self-contained composition in a document, page, application, or site
<aside></aside> Aside element Represents that is indirectly related to the content of the main content
<footer></footer> footer element Includes information about the author, copyrights and related links
<header></header> Header element It can contain the Title with logo, search bar, and the navigation aid
<h1to6></h1to6> Heading element Headings to structure the content
<main></main> Main element It contains the main content of the page and its topic
<nav></nav> Navigation element It contains the navigation bar used to navigate the pages and sections of the site
<section></section> Section element Usually used when the element mentioned above are not relevant
